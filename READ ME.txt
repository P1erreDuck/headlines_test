####### Настройки и запуск #######

Для работы необходимо заполнить 'config.py' и 'docker-compose.yml' для запуска через Docker

Для запуска через Docker в корневой папке 'headlines_test' выполняем команду 'docker-compose up -d --build'
Для локального запуска 'bot.py' (снимаем '''_''' в /database/models/build.py с 12-14 строчек, чтобы bd не перестраивалась при запуске)

Для начала работы: /start

####### Описание проекта #######
Telegram bot с функцией получения заголовков новостей из российских и зарубежных новостных источников.
Поиск заголовков осуществляется каждые 5 минут. Старые заголовки удаляются из базы данных.
1)News for you (Подборка новостей)
    Подборка из 5 случайных новостей, формирующаяся из подписок пользователя.
    Если подписок нет -  уведомление об отсутствии активных подписок и перенаправление на вкладку выбора ресурсов.
    Скриншоты работы бота/Подборка новостей.png
2)All sources (Все источники новостей)
    Перечень доступных для подписки новостных ресурсов.
    При нажатии на кнопку с названием сайта выводится список из 10 последних новостей со ссылкой на полную статью.
    Под списком размещается кнопка "Subscribe" или "Unsubscribe", в зависимости от текущего статуса подписки пользователя.
    Скриншоты работы бота/Все источники новостей.png

    Если подписка активна бот отправляет заголовки со ссылками на полную новость.
    Скриншоты работы бота/Лента подписок.png

3)Contact support
    Перенаправляет в личные сообщения к создателю бота
